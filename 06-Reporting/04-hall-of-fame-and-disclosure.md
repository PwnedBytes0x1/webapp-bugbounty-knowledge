# Hall of Fame & Responsible Disclosure

## Hall of Fame Navigation

### Types
- **Wall of Fame**: Researcher names/avatars on program website
- **Swag**: T-shirts, stickers, hoodies
- **Cash bounties**: $50 (low) to $100,000+ (critical on major programs)
- **Invite-only programs**: Access after demonstrated skill

### Getting Noticed
- Report quality matters more than quantity
- Submit critical findings on high-profile targets
- Maintain professional communication
- Follow disclosure timelines

## Responsible Disclosure

### Standard Disclosure Timeline
1. **Day 0**: Report submission
2. **Day 1-5**: Acknowledgment + triage
3. **Day 30-90**: Fix deployed (depends on severity)
4. **Day 30-90+**: Public disclosure allowed (varies)

### Disclosure Platforms
- Write on personal blog/Medium
- Open CVE request via MITRE
- Publish to VulnDB, Exploit-DB

### Disclosure Etiquette
- Follow program disclosure policy
- Give reasonable notice before public disclosure
- No full exploit code in public disclosure (concept only)
- Coordinate with program/CERT/CC
- Credit the program for patching (builds relationships)

## CVE Process

### When to Request a CVE
- Unique vulnerability (not a config issue)
- Affects multiple users/organizations
- Has a clear fix/mitigation
- Program agrees it's a valid finding

### Request via MITRE
```
https://cveform.mitre.org/
```
Requires: affected product, vulnerability description, references, researcher name

## Building Your Reputation

### Platform Rankings
- HackerOne: Reputation points, Hall of Fame tiers
- Bugcrowd: Reputation + Researcher Level
- Intigriti: Researcher levels with perks
- Synack: Red Team ranking, access to exclusive targets

### Career Path
1. Beginner (nag/basic vulns) -> VDP programs
2. Intermediate (logic bugs, chaining) -> Paid programs
3. Advanced (auth bypass, deserialization, cloud) -> Private/Invite-only programs
4. Expert (novel techniques, 0-days) -> Live hacking events, consulting
