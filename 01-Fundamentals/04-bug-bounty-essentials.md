Platform Fees (annual 2026): VDP $10K-$75K, Private $30K-$100K, Public $75K-$200K+. Synack uses flat-fee credits model.

Payouts by Severity: Critical (9.0-10.0) $3.5K-$200K+, High (7.0-8.9) $1.5K-$10K, Medium (4.0-6.9) $500-$3K, Low (0.1-3.9) $100-$500.
HackerOne paid $81M in bounties FY2025 (+13% YoY).

## Program Types
- Public: Open to all, high competition
- Private: Invite-only, less competition, higher payouts
- VDP: No rewards, CISA BOD 20-01 compliance, EU CRA compliance
- VDP Standardization: security.txt, 30d acknowledgement, safe harbor, SBOM

## Disclosure Timeline: Submission (Day 0) -> Triage (1-5d) -> Validation (1-14d) -> Bounty (2-8wk) -> Fix (30-90d) -> Public (30-90d post-fix)

## Response Labels: Triaged, N/A, Informative, Duplicate, Out of Scope, Spam, Needs More Info

## Reputation Tiers
1. Beginner: 0-5 valid, public only
2. Intermediate: 5-20 valid, starting to get invites
3. Advanced: 20-100 valid, regular private access
4. Elite: 100+ valid, top-tier private programs

## Legal: Read scope document, safe harbor essential, CFAA/GDPR/ICN Act awareness, tax implications for international payments.

## Common Beginner Mistakes
1. Testing out-of-scope assets
2. Using scanners without understanding impact
3. Writing poor reports
4. Publicly disclosing before coordinated window
5. Focusing only on critical bugs (medium bugs build reputation)
6. Not reading scope document
