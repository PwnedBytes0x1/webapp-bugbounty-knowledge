# Bug Bounty Introduction

## What is Bug Bounty?

Bug bounty is a crowdsourced security testing model where organizations reward researchers for finding and reporting security vulnerabilities.

## Major Platforms

| Platform | Focus | Payout Range |
|----------|-------|-------------|
| HackerOne | General web/mobile | $500 - $50,000+ |
| Bugcrowd | General web/mobile | $250 - $25,000+ |
| Intigriti | European targets | $200 - $10,000+ |
| Immunefi | Crypto/Web3 | $1,000 - $2,000,000+ |

## Common Mistakes

- Not reading scope carefully
- Testing out-of-scope assets
- Reporting self-XSS as critical
- Poor report quality
- Giving up too early
