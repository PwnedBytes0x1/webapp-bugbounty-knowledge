## Topics Covered

### Reconnaissance
- Passive & active recon, subdomain enumeration, tech stacking, JS analysis, cloud enumeration, social recon, content discovery

### Vulnerabilities (OWASP Top 10 + Beyond)
- XSS, SQLi, NoSQLi, SSRF, IDOR, Authentication bypass, Business logic, API security, File upload, SSTI, XXE, Command injection, CORS, Subdomain takeover, Prototype pollution, LLM/AI security, Request smuggling, OWASP Top 10 2025

### Tools
- Recon tools (subfinder, amass, httpx, naabu), scanning tools (nuclei, sqlmap), exploitation tools (Burp extensions, Frida), automation frameworks

### Advanced Topics
- Cloud security (AWS, GCP, Azure), Mobile testing, Web3/blockchain, GraphQL, JWT attacks

### Reporting
- Report writing, PoC creation, collaboration/remediation, Hall of Fame & disclosure

## Target Audience
- Intermediate to advanced bug bounty hunters
- Penetration testers specializing in web applications
- Security researchers transitioning to bug bounty

## Usage
Use as a reference during hunts, for checklists, or as a learning curriculum. Each file is self-contained with exploitation-focused content.
