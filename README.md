# Web Application Bug Bounty Knowledge Base

Comprehensive, actionable reference for web application bug bounty hunting.

## Structure
```
01-Fundamentals/       - HTTP, web tech, OWASP, headers
02-Reconnaissance/     - Passive/active recon, subdomain, content disc, JS
03-Vulnerabilities/    - XSS, SQLi, NoSQLi, SSRF, IDOR, Auth, Logic, API, etc.
04-Tools/              - Recon, scanning, exploitation, automation
05-Advanced-Topics/    - Cloud, mobile, web3, GraphQL, JWT
06-Reporting/          - Templates, severity, CVSS
07-Resources/          - Books, labs, communities, cheatsheets
```
