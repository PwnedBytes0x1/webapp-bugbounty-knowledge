> A comprehensive, community-driven knowledge base for web application bug bounty hunting. Covers everything from foundational concepts to advanced exploitation techniques, with real-world examples, tool references, and best practices.

---

## Structure

```
01-Fundamentals/       -- Core concepts, ethics, mindset, learning paths
02-Reconnaissance/     -- Subdomains, DNS, content discovery, JS analysis, cloud assets
03-Vulnerabilities/    -- OWASP Top 10, XSS, SQLi, SSRF, IDOR, API, auth, business logic
04-Tools/              -- Recon scanners, exploitation frameworks, automation
05-Advanced-Topics/    -- Cloud, mobile, Web3, GraphQL, JWT
06-Reporting/          -- Report writing, CVSS, PoC creation, triage
07-Resources/          -- Platforms, labs, wordlists, books, channels
```
