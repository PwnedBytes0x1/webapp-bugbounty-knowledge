# Communities & Platforms

## Bug Bounty Platforms

| Platform | Payout Style | Notable Programs | Notes |
|----------|-------------|------------------|-------|
| HackerOne | VDP + Bounty | Google, Twitter, Shopify, GitLab | Largest community, reputation system |
| Bugcrowd | VDP + Bounty | Tesla, Atlassian, Pinterest | Strong VRT, researcher levels |
| Intigriti | Bounty | NATO, KPN, Intel | EU-focused, monthly challenges |

## Communities

### Discord / Telegram
- **TheBountyHackers** (Discord): Active community, daily discussions
- **Bounty Hunters Co-op** (Discord): Private group for experienced hunters
- **WebSec** (Discord): General web security chat
- **Bug Bounty World** (Telegram): Urdu/Hindi community
- **Bug Bounty Forum** (Telegram): Global community

### Reddit
- r/bugbounty: General discussion
- r/netsec: Security news
- r/AskNetsec: Questions & advice
- r/HowToHack: Learning resources

### Forums
- **Bugcrowd Forum**: Program announcements, researcher discussions
- **HackerOne Community**: Activity feed, program updates
- **0x00sec.org**: Malware/hacking forum
