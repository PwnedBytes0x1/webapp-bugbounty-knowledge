# Labs & Courses

## Free Labs

| Lab | URL | Focus |
|-----|-----|-------|
| **PortSwigger Web Security Academy** | https://portswigger.net/web-security | 200+ labs, all OWASP categories |
| **Hacker101** | https://www.hacker101.com | Video course + CTF |
| **PentesterLab** | https://pentesterlab.com | Free exercises + PRO |

| **Damn Vulnerable Web App (DVWA)** | http://www.dvwa.co.uk | Classic LAMP vulnerable app |

| **TryHackMe** | https://tryhackme.com | Beginner friendly, free rooms |

## Recommended Learning Path

```
1. PortSwigger Academy (all labs) — Free
2. Hacker101 CTF — Free
3. TryHackMe Web Path — ~$10/month
4. HackTheBox Starting Point + Easy boxes — ~$10/month
5. PentesterLab Pro — $19/month
6. Live targets (HackerOne/Bugcrowd public programs)
```
