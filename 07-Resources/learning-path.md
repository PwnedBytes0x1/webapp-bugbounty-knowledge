# Learning Path

## Beginner Path (Months 1-6)

### Month 1-2: Foundations
- HTTP/HTTPS protocol fundamentals
- Web technologies (HTML, CSS, JavaScript, APIs)
- Basic Linux command line
- Networking basics (DNS, TCP/IP, ports)
- PortSwigger Web Security Academy

### Month 3-4: Core Skills
- Burp Suite basics (Proxy, Repeater, Intruder)
- OWASP Top 10 vulnerabilities
- XSS, SQLi, IDOR hands-on labs
- TryHackMe, HackTheBox

### Month 5-6: Methodology and Recon
- Complete bug bounty methodology
- Reconnaissance tools and techniques
- Subdomain enumeration, JS analysis
- Content discovery, fuzzing

## Intermediate Path (Months 7-12)

- Business logic vulnerabilities
- API security (REST, GraphQL)
- Auth/authorization testing
- SSRF, SSTI, XXE deep dives
- Cloud security basics
- Bug bounty platform registration
- First bounties
