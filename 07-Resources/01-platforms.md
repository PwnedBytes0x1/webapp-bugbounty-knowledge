# Platforms & Programs

## Major Bug Bounty Platforms

| Platform | URL | Features |
|----------|-----|----------|
| **HackerOne** | https://hackerone.com | Largest, gov programs, live hacking events |
| **Bugcrowd** | https://bugcrowd.com | Strong VDP program, managed bug bounties |
| **Intigriti** | https://intigriti.com | EU-focused, 1337Up CTF, monthly challenges |
| **YesWeHack** | https://yeswehack.com | European, edu programs |
| **Synack** | https://synack.com | Invite-only, hourly rates + bounties |
| **SafeHats** | https://safehats.com | Indian platform, various programs |

## Vulnerability Disclosure Programs (VDP)

- **Google VDP** — https://bughunters.google.com
- **Microsoft Bounty Programs** — https://www.microsoft.com/en-us/msrc/bounty
- **Apple Security Bounty** — https://security.apple.com/bounty/
- **Meta Bug Bounty** — https://bugcrowd.com/facebook
- **OpenAI Bug Bounty** — https://bugcrowd.com/openai
- **Cloudflare** — https://hackerone.com/cloudflare

## Live Hacking Events

- **H1-XXXX** — HackerOne live events (e.g., H1-702, H1-NASA)
- **Bugcrowd Bashes** — Time-boxed events with bonuses
- **Intigriti 1337Up** — Annual CTF with real-world challenges

## Private Program Access

To get invited:
1. Build reputation on public programs (valid reports)
2. Submit high-quality reports
3. Be active in the community
4. Use the platform's request feature
