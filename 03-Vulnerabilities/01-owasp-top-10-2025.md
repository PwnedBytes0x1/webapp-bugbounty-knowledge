# OWASP Top 10 2025: Bug Bounty Perspective

The OWASP Top 10 is not a vulnerability checklist — it is a prioritization framework. For bug bounty, the relevant question is not "is this in the Top 10" but "does this provide exploitable impact on this specific target."

## A01:2025 – Broken Access Control

The most common critical finding in bug bounty. Includes IDOR, privilege escalation, and missing function-level checks.

**Hunter focus:** Every authenticated endpoint is a candidate. Test with another user's session, test with modified role claims, test with direct object references.

## A02:2025 – Cryptographic Failures

Weak TLS, expired certs, padding oracle, predictable RNG, hardcoded keys.

**Hunter focus:** JWT algorithm confusion, padding oracle in encryption oracles, weak password hashing, predictable session tokens.

## A03:2025 – Injection

SQLi, NoSQLi, OS command injection, SSTI, XXE.

**Hunter focus:** SSTI in email templates and report generators. XXE in file upload handlers (Office formats). Command injection in ping/curl/dig features. NoSQL injection in JSON endpoints with MongoDB backends.

## A04:2025 – Insecure Design

Business logic flaws, rate limiting bypass, missing security controls in design.

**Hunter focus:** This is where the highest payouts live. Coupon abuse, account creation bypass, multi-step flow tampering, race conditions.

## A05:2025 – Security Misconfiguration

Default credentials, unnecessary features enabled, verbose error messages, missing security headers.

**Hunter focus:** Debug endpoints, stack traces, CORS misconfigurations, directory listing, security header gaps.

## A06:2025 – Vulnerable Components

Outdated libraries/frameworks with known CVEs.

**Hunter focus:** Use Wappalyzer/BuiltWith for fingerprinting, then cross-reference with CVEs. Older versions of jQuery, Lodash, Spring, Express, Django are frequent winners.

## A07:2025 – Identification & Authentication Failures

Weak passwords, credential stuffing, session fixation, missing MFA.

**Hunter focus:** OAuth misconfiguration, JWT weaknesses, session fixation, password reset token flaws, rate limiting bypass on auth endpoints.

## A08:2025 – Data Integrity Failures

CSP bypass, unsafe deserialization, CI/CD pipeline compromises.

**Hunter focus:** Unsafe deserialization in Java/Python/PHP apps. CSP bypass with JSONP endpoints, CDN subdomain takeover, npm dependency confusion.

## A09:2025 – Monitoring & Logging Failures

Insufficient logging for attack detection.

**Hunter focus:** Harder to exploit directly but enables chaining. No rate limit logging = undetected credential stuffing. No audit log = undetected IDOR abuse.

## A10:2025 – SSRF

Server-side request forgery.

**Hunter focus:** Cloud metadata extraction, internal port scanning, pivot to internal services. SSRF is the modern RCE-equivalent in cloud environments.
